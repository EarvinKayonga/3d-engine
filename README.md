### Setup

#### git clone and build

```
git clone this repo
cd 3d-engine
make
```

#### Run the project

```
//if the build passes
./Bin
```

#### Makefile rules

```
make or make re // to build project
make fclean  	// to remove the *.o and others binaries
make clean	// to remove *.o
```
